## Objectif

---

## 1️⃣ Qu’est-ce que Google Cloud ?

Google Cloud, c’est la plateforme cloud de Google. Elle permet aux entreprises de :

*   **Héberger leurs données et leurs applications** de manière sécurisée.
*   **Analyser de grandes quantités de données** avec BigQuery.
*   **Utiliser l’IA et le machine learning** via Vertex AI.
*   **Créer des machines virtuelles** avec Compute Engine.
*   **Stocker des fichiers partout dans le monde** avec Cloud Storage.

En résumé : Google Cloud aide les entreprises à **aller plus vite, réduire leurs coûts et sécuriser leurs données**.
